# 数据分析脚本和文档

本目录包含数据分析相关的所有脚本和文档。

## 脚本文件

### Action数据格式分析

- **analyze_action_format.py** - 分析action数据格式
- **check_action_format.py** - 检查action数据格式
- **check_action_info.py** - 检查action信息

### 数据集结构分析

- **analyze_dataset_structure.py** - 分析数据集结构

### 变换和旋转分析

- **analyze_rotation_representation.py** - 分析旋转表示
- **analyze_transform_relationship.py** - 分析变换关系
- **verify_transform_relationship.py** - 验证变换关系

## 文档

- **action_data_format_analysis.md** - Action数据格式分析文档

## 使用说明

这些脚本主要用于分析BridgeData V2数据集的结构和格式。

详细说明请参考 [action_data_format_analysis.md](action_data_format_analysis.md)
