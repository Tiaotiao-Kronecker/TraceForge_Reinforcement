# Repository Guidelines

This repository keeps its shared contributor and agent instructions in `CLAUDE.md`.

## Source of Truth
For project context, environment setup, core commands, architecture notes, testing guidance, and debugging conventions, read `CLAUDE.md` first and treat it as the primary maintained guide for this repository.

## Maintenance Rule
To avoid duplicated documentation, update `CLAUDE.md` when workflow or repository guidance changes. Keep `AGENTS.md` minimal and only adjust it if the delegation rule or document location changes.

## Precedence
If `AGENTS.md` and `CLAUDE.md` ever differ, follow `CLAUDE.md`.
